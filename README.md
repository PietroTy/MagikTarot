# Mística

Plataforma de consultas místicas com IA — tarot, mapa astral, numerologia, horóscopo e muito mais.

## Rodar localmente

### 1. Backend (porta 3001)
```bash
cd backend
npm install
node server.js
```

### 2. Frontend (porta 3002)
```bash
npm install
npm start
# Acesse: http://localhost:3002
```

### 3. Modo completo com tunnel (deploy)
```bash
./deploy.sh
```
